# 🚀 水印功能快速参考

## ⚡ 一分钟上手

```
访问地址: http://localhost:3001/tools/
导航到: 加水印 模块
```

## 📁 支持的格式

| 格式 | 扩展名 | 处理方式 | 水印效果 |
|------|--------|----------|----------|
| **PDF** | .pdf | pdf-lib | 平铺水印，每页覆盖 |
| **图片** | .jpg, .png, .bmp, .webp, .gif | Canvas | 透明平铺水印 |
| **Word** | .docx | mammoth + docx | 段落间插入水印 |

## 🎛️ 参数设置

| 参数 | 范围 | 默认值 | 推荐值 |
|------|------|--------|--------|
| 水印文本 | 任意 | "水印" | 8-20字 |
| 透明度 | 10%-100% | 30% | 20-40% |
| 字体大小 | 24-120px | 24px | 36-48px |
| 旋转角度 | -90°-90° | -45° | -45° 或 45° |

## 🐛 常见问题

**Q: 水印看不清？**
A: 增加字体大小，降低透明度

**Q: 水印太明显？**
A: 增加透明度到 40% 以上

**Q: 处理很慢？**
A: 正常现象，本地处理确保安全

**Q: 中文显示异常？**
A: 已自动处理，会转为图片
