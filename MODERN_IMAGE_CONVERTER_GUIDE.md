# 🚀 现代图片格式转换器 - 快速使用指南

## ✨ 功能特性

### 🔒 本地处理
所有图片转换在浏览器中完成，文件不上传到服务器，保护隐私安全。

### 👀 实时对比
支持原图和转换后图片的并排对比，滑动查看压缩前后的视觉差异。

### 🎚️ 质量控制
精细的质量滑块（40-100%），在文件大小和图片质量之间找到完美平衡。

### 📦 批量导出
支持批量转换和 ZIP 打包下载，保持一致的质量设置。

## 📋 支持格式

| 输入格式 | 输出格式 | 浏览器要求 |
|---------|---------|-----------|
| AVIF | ✅ AVIF, WebP, PNG, JPG | Chrome 90+, Edge 90+, Firefox 93+ |
| WebP | ✅ AVIF, WebP, PNG, JPG | Chrome 23+, Firefox 65+, Edge 18+ |
| PNG | ✅ AVIF, WebP, PNG, JPG | 所有浏览器 |

## 🎯 使用步骤

### 步骤 1：上传文件 📤
1. 点击上传按钮
2. 选择 AVIF / WebP / PNG / JPG 文件
3. 支持拖拽上传
4. 支持批量上传多个文件

### 步骤 2：选择输出格式 🎨
选择目标格式：
- **AVIF**: 最佳压缩（推荐质量 40-60）
- **WebP**: 平衡性能（推荐质量 75-85）
- **PNG**: 无损压缩（不需要设置质量）
- **JPG**: 通用格式（推荐质量 80-90）

### 步骤 3：调整质量 🎚️
- 拖动质量滑块（40-100%）
- PNG 格式无需调整质量
- 较低质量 = 更小文件
- 较高质量 = 更好画质

### 步骤 4：开始转换 🎬
1. 点击"开始转换"按钮
2. 查看实时进度条
3. 等待转换完成
4. 查看压缩比例

### 步骤 5：对比和下载 💾

**单个文件操作：**
- 点击 🎚️ 按钮：打开并排对比视图
- 拖动中间滑块查看压缩前后差异
- 点击 💾 按钮：下载单个文件

**批量下载：**
- 点击"打包下载 ZIP"按钮
- 自动打包所有转换后的图片
- 一次下载所有文件

## ⚠️ 注意事项

### 浏览器兼容性
- **AVIF**: 需要 Chrome 90+, Edge 90+, Firefox 93+
- **WebP**: 需要 Chrome 23+, Firefox 65+, Edge 18+
- 如果浏览器不支持 AVIF，系统会自动提示

### 透明通道处理
- PNG → JPG: 自动添加白色背景
- PNG → WebP/AVIF: 保留透明通道
- JPG → PNG/WebP/AVIF: 无透明通道

### 文件大小建议
- 单张图片建议 < 30MB
- 批量转换建议 < 20 张图片
- 移动设备建议单张 < 10MB

## 🔧 故障排除

### 问题：AVIF 不支持
**解决方案：**
- 升级浏览器到 Chrome 90+ 或 Edge 90+
- 或选择 WebP 格式作为替代

### 问题：转换失败
**可能原因：**
- 图片文件损坏
- 浏览器内存不足
- 文件格式不支持

**解决方案：**
- 检查图片是否能正常打开
- 关闭其他标签页释放内存
- 尝试其他格式

### 问题：质量滑块无效
**原因：**
- PNG 格式是无损的，不支持质量调整

**解决方案：**
- 选择 AVIF、WebP 或 JPG 格式

## 🌟 最佳实践

### 网站优化
1. 照片用 AVIF（最小文件）
2. 设计图用 WebP（兼容性好）

### 质量平衡
- 重要图片：质量 85-95
- 普通图片：质量 75-85
- 背景图：质量 60-75
- 缩略图：质量 50-70
