# 🎯 CommonTools 功能总结

## 📱 已实现的功能

### 1. 文件加密/解密
- 支持多种文件格式
- AES-256 加密
- 密码强度检测
- 本地处理，隐私保护

### 2. 格式转换
- PDF ↔ Word
- Markdown → PDF
- 多种文档格式互转

### 3. PDF 水印
- 支持 PDF 和图片
- 中文水印支持
- 预览后下载
- 自定义透明度、大小、角度

### 4. 电子签名
- 手写签名
- 签名保存和管理
- 添加到 PDF

### 5. 文件压缩/解压
- ZIP 格式支持
- 批量处理
- 进度显示

### 6. HEIC 转 JPG
- 批量转换
- 质量可调
- 预览功能

### 7. **🆕 Live Photo 转换**（新功能）
- **静态图片**：HEIC → JPG
- **GIF 动图**：MOV → GIF（调色板优化）
- **MP4 视频**：MOV → MP4（H.264 编码）
- **智能帧去重**：减小文件大小
- **高级设置**：质量、帧率、分辨率可调
- **进度显示**：实时反馈处理状态
- **本地处理**：隐私保护
