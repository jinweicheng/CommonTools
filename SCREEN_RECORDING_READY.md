## 🔧 技术亮点

### 1. FFmpeg.wasm 完整集成 ⭐⭐⭐⭐⭐

```typescript
// 裁剪
filters.push(`crop=${w}:${h}:${x}:${y}`)

// 压缩
args.push('-c:v', 'libx264', '-crf', '23')

// 模糊
filters.push(`boxblur=10:1`)

// 组合命令
await ffmpeg.exec(['-i', 'input.mp4', '-vf', filters.join(','), ...args, 'output.mp4'])
```

### 2. 智能错误处理 ⭐⭐⭐⭐⭐

// 格式检测

  setError('不是视频文件')
}
