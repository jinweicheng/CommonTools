## ⚡ 快速开始

### 1. 访问应用
```
http://localhost:3001/tools/
点击: 加水印
```

### 2. 设置参数
```
水印文本: 输入你的水印（如 "机密"）
透明度: 30% (推荐 20-40%)
字体大小: 48px (推荐 36-60px)
角度: -45° (推荐 -45° 或 45°)
```

### 3. 上传文件
```
点击按钮 → 选择 PDF 或图片 → 自动处理 → 下载结果
```

## ⚠️ 不支持其他格式

### 仅支持 PDF 和图片

本工具**专注于 PDF 和图片格式**，不支持 Word/Excel/PPT 等其他格式。

**为什么？**
- ✅ **技术可靠**：PDF 和图片的水印技术成熟
- ✅ **效果专业**：真正的透明平铺背景水印
- ✅ **不误导用户**：明确功能边界，避免期望落差

### 其他格式如何加水印？

**Office 文档**（Word/Excel/PPT）：
- 使用软件自带的水印功能
- Word：设计 → 水印
- Excel：页面布局 → 背景
- PowerPoint：设计 → 格式背景

## 🐛 常见问题

**Q: 水印看不清楚？**
```
解决: 增大字体 (48-60px) + 降低透明度 (20-30%)
```

**Q: 水印太明显遮挡内容？**
```
解决: 提高透明度 (40-50%) + 减小字体 (28-36px)
```

**Q: 中文显示异常？**
```
解决: 已自动处理，会转为图片，无需担心
```

**Q: 需要给 Word 加水印？**
```
解决: Word → 另存为 PDF → 使用本工具
```

**Q: 处理速度慢？**
```
说明: 正常现象，本地处理保证安全，速度取决于文件大小
```

## 🚀 高级技巧

### 1. 最佳透明度选择
```
机密程度高 → 透明度低 (20-30%) → 水印明显
机密程度中 → 透明度中 (30-40%) → 平衡效果
仅作标识   → 透明度高 (40-50%) → 水印淡雅
```

### 2. 角度选择建议
```
-45° → 经典水印角度，最常用
45° → 反向倾斜，别具一格
0° → 水平排列，适合简短文本
```

### 3. 字体大小选择
```
24-36px → 适合长文本
36-48px → 标准大小，通用
48-72px → 适合短文本或强调
72-120px → 大标识，醒目明显
```
