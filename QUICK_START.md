# Live Photo 转换功能 - 快速开始

## 🚀 当前状态（立即可用）

Live Photo 转换功能**已完成并可用**！

### ✅ 可用功能

| 功能 | 状态 | 说明 |
|------|------|------|
| **HEIC → JPG** | ✅ 完美 | 静态照片转换 |
| **MOV → GIF** | ✅ 可用 | 动画 GIF（推荐） |
| **MOV → MP4** | ⚠️ 受限 | 需要 FFmpeg（可能失败） |

### 🎯 推荐使用

**MOV → GIF 转换**（无需任何配置，立即可用）

## 📝 使用步骤

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 访问页面

打开浏览器访问：`http://localhost:3000/tools/live-photo`

### 3. 转换文件

1. 点击"上传 MOV"按钮
2. 选择您的 `.MOV` 文件
3. 选择转换模式：**GIF**（推荐）
4. 点击"转换"按钮
5. 等待 5-15 秒
6. 点击"下载"保存 GIF

### 4. 完成！

就是这么简单！

## ⚙️ 可选配置

### GIF 质量设置

点击"高级设置"可调整：

- **质量**（1-20）：默认 10
  - 越小 = 越好（但文件更大）
  - 建议：5-15

- **帧率**（5-30 FPS）：默认 10
  - 越高 = 越流畅（但文件更大）
  - 建议：8-12

- **宽度**（240-1080px）：默认 480
  - 越大 = 越清晰（但文件更大）
  - 建议：320-640

### 推荐配置

**平衡模式（默认）：**
- 质量：10
- 帧率：10 FPS
- 宽度：480px
- 结果：约 2-5 MB

**高质量模式：**
- 质量：5
- 帧率：15 FPS
- 宽度：640px
- 结果：约 5-10 MB

**小文件模式：**
- 质量：15
- 帧率：8 FPS
- 宽度：320px
- 结果：约 1-2 MB

## 🔧 故障排除

### 问题：转换失败

**解决方法：**
1. 确保文件是有效的 MOV 格式
2. 尝试使用不同的浏览器（推荐 Chrome）
3. 刷新页面重试

### 问题：转换很慢

**解决方法：**
1. 降低帧率（8-10 FPS）
2. 减小宽度（320-480px）
3. 等待完成（10秒视频约需 10-20秒）

### 问题：文件太大

**解决方法：**
1. 增加质量值（10-15）
2. 降低帧率（8 FPS）
3. 减小宽度（320px）
