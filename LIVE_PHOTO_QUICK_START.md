# 🚀 Live Photo 转换工具 - 快速开始

## 3步快速转换

### 步骤 1：上传文件

1. 点击"上传 HEIC 图片"或"上传 MOV 视频"
2. 从 iPhone 导出的 Live Photo 文件
3. 根据需求上传一个或两个文件

### 步骤 2：选择模式

- **静态图片** - 仅需 HEIC → JPG
- **GIF 动图** - 仅需 MOV → GIF（推荐微信分享）
- **MP4 视频** - 仅需 MOV → MP4（推荐高质量保存）

### 步骤 3：开始转换

1. 点击"开始转换"
2. 等待处理完成（10-30秒）
3. 点击"下载结果"

---

## 推荐设置

### 微信/QQ 分享

```
模式：GIF
帧率：10fps
宽度：480px
质量：10
```

**效果**：
- 文件大小：1.5-3MB
- 清晰度：适中
- 兼容性：完美

### 高质量保存

```
模式：MP4
质量（CRF）：23
帧去重：启用
阈值：5%
```

**效果**：
- 文件大小：500KB-1.5MB
- 清晰度：高
- 支持音频：是

## 常见场景

### 场景 1：分享到微信群

1. 上传 MOV 文件
2. 选择"GIF 动图"模式
3. 设置：帧率 10fps，宽度 480px
4. 转换并下载
5. 直接发送到微信

### 场景 2：保存到相册

1. 上传 MOV 文件
2. 选择"MP4 视频"模式
3. 启用帧去重
4. 转换并下载
5. 保存到本地相册

### 场景 3：仅需静态图

1. 上传 HEIC 文件
2. 选择"静态图片"模式
3. 转换并下载
4. 得到高质量 JPG

## 获取 Live Photo 文件

### iPhone 导出方法

#### 方法 1：AirDrop（推荐）

1. iPhone 打开"照片"应用
2. 选择实况照片
3. 点击分享图标
4. 选择 AirDrop
5. 发送到 Mac/PC
6. 会得到 HEIC + MOV 两个文件

#### 方法 2：文件应用

1. iPhone 打开"文件"应用
2. 找到照片文件夹
3. 选择实况照片
4. 复制到 iCloud Drive
5. 在 PC 上访问 iCloud.com 下载

#### 方法 3：iTunes/Finder

1. iPhone 连接到电脑
2. 打开 iTunes（Windows）或 Finder（Mac）
3. 选择照片同步
4. 导出实况照片
5. 会得到 HEIC + MOV 两个文件

## 注意事项

1. **文件大小**：推荐 MOV < 50MB（避免浏览器卡顿）
2. **网络要求**：首次使用需下载 FFmpeg（约30MB）
3. **内存要求**：推荐可用内存 > 2GB
4. **隐私保护**：所有处理本地完成，不上传服务器

## 支持的格式

### 输入

- ✅ HEIC（iPhone 照片）
- ✅ HEIF（HEIC 的标准格式）
- ✅ MOV（iPhone 视频）
- ✅ MP4（通用视频）

### 输出

- ✅ JPG（静态图片，兼容性最好）
- ✅ GIF（动图，自动循环）
- ✅ MP4（视频，画质最好）
